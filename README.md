## Проекты
### Loglint — Go static analyzer for log messages
https://github.com/vladik2801/loglint

- Реализовал статический анализатор на базе `go/analysis` (AST-анализ кода)
- Проверяет текст лог-сообщений по набору правил (формат, спецсимволы, sensitive данные)
- Поддерживает конфигурацию через JSON
- Добавлены SuggestedFixes для автоматического исправления нарушений
- Интегрирован в `golangci-lint` как module plugin
- Настроен CI (GitHub Actions) для сборки и тестирования
- Протестирован на реальных open-source проектах
